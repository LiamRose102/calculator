Simple Calculator

This application performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

To run the app:
1. Clone the repository
2. Build the Docker image: docker build -t my-calculator .
3. Run the container: docker run my-calculator (Not sure why its giving me an error the code should work)
